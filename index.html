what is node.js?
simply put; node.js is a javascript interpreter
node.js =
v8 (javascript engine) + event loop + I/O API
v8 is Chrome's javascript compiler
an event loop is a circular queue that dispatches messages
node.js using its I/O API
can read and write files on the filesystem
node.js inherits javascript's asynchronous nature 
fs.readFile('start.html', function (err, data) {
  console.log('flag 1');
  if (!err) {
    console.log('file contents:', data);
  } else {
    console.log('error:', err);
  }
});

console.log('flag 2');

var handlerFunction = function (err, data) {
  console.log('flag 1');
  if (!err) {
    console.log('file contents:', data);
  } else {
    console.log('error:', err);
  }
}

fs.readFile('start.html', handlerFunction);

console.log('flag 2');
node.js is mainly used for backend web development
npm (node.js package manager) is the most used package manager for javascript libraries and has become a major player on frontend web development
since node.js executes code, it can do anything code can do
from web dev to desktop applications and command line utilities
